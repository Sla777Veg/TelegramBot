# Telegram bot. Animal Shelter. 
## Description of the project:

Telegram bot allows you to serve clients of an animal shelter, providing them with all the necessary information about the possibilities for adopting an animal. 
Also, providing an opportunity for contact between the client and the shelter administrator, thus providing information to the administrators about the client and his responsibility in caring for the pet.

## The technologies used in the project:
* Java 17
* Maven
* Stream API
* Spring Boot
* Spring JPA
* Hibernate
* PostgreSQL
* Liquibase
